

## Compilation

```
mkdir build
cd build
cmake ..
make
```

# Testing environment (visual):
## Dependences

- opencv ( https://opencv.org/ )

install opencv from repository (https://github.com/opencv/opencv/releases) or you OS package manager e.g.:
```
pamac install opencv
```
